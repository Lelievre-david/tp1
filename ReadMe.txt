### Format des fichiers de tests : 

- première ligne deux entiers strictement positifs : l et h
- deuxième ligne un entier : le nombre de points n>=0
- ensuite un point par ligne (x puis y), les points étant triés
  par x croissants.  

Le nom du fichier indique le nombre de points et la valeur attendue.
Par exemple pour ex_N500_res7854 : 500 points, valeur attendue 7854.

ex=* et exT* indiquent que les données ont une propriété particulière
- regardez les fichiers - qui peut eventuellement influencer le temps
d'exécution de votre algorithme, selon l'algorithme.
